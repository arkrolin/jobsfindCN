# 目标公司 → 算法/八股文高频考点映射

## 使用说明

此文件定义了不同 tier 的公司面试中，算法主题和八股文分类的出现频率和重要性。用于：
1. **算法计划**：帮助确定哪些算法主题需要优先刷
2. **八股文计划**：帮助确定哪些八股文分类需要优先背诵

频率标记：
- 🔴 **必考**：该司几乎所有面试都会出现
- 🟡 **高频**：大部分面试会出现
- 🟢 **常见**：有一定概率出现
- ⚪ **偶见**：少数面试会出现

---

## Tier 1: 一线大厂

### 字节跳动
**算法侧重：**
- 🔴 动态规划（背包、子序列、路径）
- 🔴 数组/字符串（实际业务场景模拟题）
- 🟡 链表（反转、K个一组翻转）
- 🟡 树/二叉树（非递归遍历、序列化）
- 🟡 哈希表
- 🟢 贪心
- ⚪ 位运算

**八股文侧重（通用）：**
- 计算机网络（TCP/HTTP，字节强网络基建）
- 数据库（MySQL索引+Redis，高并发场景）
- 并发编程
- 分布式（消息队列、分布式一致性）

**面试风格：** 3-4 轮技术面，算法题偏难，喜欢出实际场景变种题。手写代码。

### 腾讯
**算法侧重：**
- 🔴 动态规划
- 🔴 树/图（腾讯游戏/社交场景）
- 🟡 数组
- 🟡 字符串
- 🟢 排序/搜索
- 🟢 链表

**八股文侧重（通用）：**
- 计算机网络（TCP/UDP/HTTP）
- 数据库（MySQL+Redis）
- 操作系统（内存管理）
- C++/Go 语言特性（腾讯主要技术栈）

**面试风格：** 2-3 轮技术面，中等难度为主。重视基础和项目经验。

### 阿里巴巴
**算法侧重：**
- 🔴 动态规划
- 🟡 树/图
- 🟡 哈希表
- 🟡 设计题
- 🟢 排序/搜索

**八股文侧重（通用）：**
- 数据库（MySQL 深究——索引、锁、事务、分库分表）
- 分布式（阿里强分布式基因）
- JVM（Java 为主流）
- 设计模式

**面试风格：** 看重工程能力和系统设计。算法中等偏难。Java 为主。

### 美团

- 🔴 动态规划
- 🟡 数组
- 🟡 链表
- 🟡 哈希表
- 🟢 树/二叉树

**八股文侧重（通用）：**
- 数据库（MySQL 深究）
- 分布式
- Java 并发编程
- Redis

**面试风格：** 2-3 轮技术面，中等难度。重视业务理解和实际问题解决。

### 拼多多
**算法侧重：**
- 🔴 动态规划
- 🟡 数组
- 🟡 字符串
- 🟢 贪心

**八股文侧重（通用）：**
- 数据库

- 分布式基础

## Tier 2: 二线/独角兽

**算法侧重：**
- 🔴 数组/字符串（中等为主）
- 🟡 哈希表
- 🟡 动态规划（中等为主）
- 🟡 树/二叉树
- 🟢 链表
- 🟢 栈/队列

**八股文侧重（通用）：**
- 计算机网络基础
- 数据库基础
- 编程语言基础
- 框架（Spring/Go框架）

**面试风格：** 2-3 轮技术面，偏向中等难度。看重实际项目经验。

## Tier 3: 外企

### Microsoft / Amazon / Zoom
**算法侧重：**
- 🔴 树/图（高频）
- 🔴 动态规划
- 🟡 设计题（LRU/LFU等）
- 🟡 数组/字符串
- 🟡 哈希表
- 🟢 链表

**八股文侧重：**
- System Design（OOD/系统设计）
- 数据结构基础
- 算法复杂度分析
- 行为面试（Leadership Principles）

**面试风格：** 结构化面试，注重沟通和思维过程。更需要英文准备。

## 算法主题总体优先级（综合 Tier 1-3）

| 优先级 | 主题 | 频率 | 说明 |
|--------|------|------|------|
| **P0** | 动态规划 | 🔴 | 几乎所有大厂必考，从 Easy 到 Hard 全覆盖 |
| **P0** | 数组/字符串 | 🔴 | 考查最频繁，实际场景模拟题多 |
| **P0** | 哈希表 | 🟡 | 常与其他主题组合考查 |
| **P1** | 树/图 | 🟡 | 大厂和外企高频 |
| **P1** | 链表 | 🟡 | 字节高频，其他公司常见 |
| **P1** | 二分查找 | 🟡 | 中等难度居多 |
| **P2** | 栈/队列 | 🟢 | 单调栈是热点 |
| **P2** | 双指针 | 🟢 | 滑动窗口高频 |
| **P2** | 排序/搜索 | 🟢 | 快速选择、Top K 常见 |
| **P2** | 贪心 | 🟢 | 策略题，出现频率中等 |
| **P3** | 回溯 | 🟢 | 排列组合，出现频率降低 |
| **P3** | 设计 | 🟢 | LRU Cache 类经典题 |
| **P3** | 位运算 | ⚪ | 偶尔出现 |
| **P3** | 数学 | ⚪ | 偶尔出现 |
